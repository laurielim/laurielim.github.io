Gotta Catch'em All!

Press start to play! Pokemons will show up where the pokeballs are. Tap on the pokemons to score points. It's game over if you tap on a pokeball.

To be the very best, you can set the difficulty in settings. In level 1, you can skip up to 5 pokemons, in level 2, only 3 and in level 3, skipping any means game over!
